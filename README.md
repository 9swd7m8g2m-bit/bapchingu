# 🍱 밥친구

> 회사 점심 밥친구 조편성을 자동으로 만들고, 맛집을 고르고, 조끼리 메뉴를 정하는 도구

**🔗 바로 쓰기: https://9swd7m8g2m-bit.github.io/bapchingu/**

---

## 왜 만들었나요?

우리 회사는 점심 때 **4~5명씩 밥친구 조**를 짜서 일주일을 같이 먹어요.
지금까지는 한 사람이 매주 월요일에 코드를 돌려 조편성 이미지를 만들어 올렸는데, 이게 은근 번거로웠어요.

- 연차·출장으로 빠지거나, 샐러드팀으로 옮기거나, 중간에 누가 합류하면 → **이미지를 매번 다시 만들어야 함**
- 그 작업을 **한 명이 전담** → 그 사람이 휴가 가면 다른 사람이 대신해야 함
- 메뉴 고르는 것도 매번 고민

그래서 **누구나, 언제든, 클릭 몇 번으로** 조편성을 만들고 수정하고, 맛집까지 같이 고를 수 있게 만들었어요.

---

## 무엇을 할 수 있나요?

### 1. 🔀 조편성
- 버튼 한 번으로 **4~5명씩 자동 조 편성**
- **팀장 공평 로테이션** — 팀장을 적게 맡은 사람부터 돌아가며 자동 선발
- **3주 연속 같은 조 방지** — 매번 새로운 사람들과 (가끔 겹침은 OK)
- 완성된 표를 **이미지로 저장** → 카톡/팀즈에 붙여넣기

### 2. 👥 멤버 관리
- 팀별 명단에서 각자 상태를 **참여 / 연차·출장 / 샐러드팀** 으로 변경 (조편성에 즉시 반영)
- 👑 버튼으로 **팀장 풀** 지정
- 중도합류자 추가

### 3. 🍽️ 엔도가이드
- 우리가 직접 만드는 **회사 주변 맛집 가이드** (미슐랭처럼)
- 별점 · 대표메뉴 · 가격대 · 도보 시간 · 한줄평 기록
- 카테고리별로 골라보기, **네이버지도 바로가기**

### 4. 💬 조별 점심 회의실
- 각 조 카드의 `💬 회의실` 버튼으로 입장
- 엔도가이드에서 **식당 후보 추천** → 👍 😋 ❤️ 👎 로 투표 → **따봉 많은 곳 1위 표시**
- 조원들끼리 **채팅** (실시간)

---

## 어떻게 쓰나요?

1. **월요일 조편성**: `🔀 조 생성` → 마음에 들면 `✅ 이번 주 확정` → `📷 이미지 저장` 해서 단톡방에 공유
2. **누가 빠지거나 추가되면**: `👥 멤버 관리`에서 상태만 바꾸면 끝 (이미지 다시 저장)
3. **메뉴 정하기**: 조 카드의 `💬 회의실`에서 식당 추천하고 투표 → 제일 많이 받은 곳으로!

> 💡 **누구나 조작할 수 있고, 모두가 같은 화면을 봅니다.** (실시간 공유)
> 실수로 지우거나 바꿔도 화면 아래 **`↩ 되돌리기`** 로 복구할 수 있어요.
